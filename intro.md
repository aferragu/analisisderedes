# Análisis de Redes

## Universidad ORT Uruguay. Facultad de Ingeniería

### Docente: Andrés Ferragut

Este curso tiene como objetivo proporcionar a los estudiantes las habilidades necesarias para analizar, modelar fenómenos estocásticos, con particular énfasis en los sistemas interconectados, como sistemas cloud, redes de datos y otros. Los participantes aprenderán a utilizar técnicas matemáticas, con fuerte foco en la probabilidad para analizar estos sistemas de manera teórica, así como herramientas de programación para simular estos sistemas y obtener indicadores de desempeño.

Se busca combinar teoría y práctica, mediante la utilización de ejemplos de sistemas reales, analizados mediante código `julia`.
